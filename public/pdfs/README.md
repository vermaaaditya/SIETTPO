# PDF Files Directory

This directory is for storing PDF files that will be displayed on the website.

## Required Files

Please upload the following PDF files to this directory:

1. **recruitment-brochure.pdf** - The recruitment brochure for the TPO
2. **recruitment-guidelines.pdf** - The recruitment guidelines for companies

## How to Upload

Simply place your PDF files in this directory with the exact filenames listed above. The website will automatically serve them from here.

## File Paths

Once uploaded, the files will be accessible at:
- `/pdfs/recruitment-brochure.pdf`
- `/pdfs/recruitment-guidelines.pdf`
